Introduction of Project:

  • We have implemented here Django project with token based Authentication. We have use Python programming Language and Django and Django rest Framework for API development.
  • We have documented All APIs and also making all APIs CRUD operations. Due to lacking of knowlegde of AI and ML we have use DB data and perform on it and do not use any trained model of ML

Project Setup and run in local or Docker:

  • First we need to install python. And make virtualenv due to avoid to install deendencies in globle and run command but insure you have in project Directory and env is activated pip install -r requirements.txt
  • When all dependancy install then migrate it for build DB and create and changes of migrations in databse. To migrate the project we need to run command python manage.py migrate I have send all migration file that's why no need to apply migration if need then first run python manage.py makemigrations after migrate the project.
  • When everything done then run cmd python manage.py runserver to run project.
  • I have build docker file and docker compose file.
  • If want to run this project on docker then we need to first install Docker CLI in aur local system. When docker is installed then firs we need to build build image in docker. To building of image we need to run docker-compose.yml file.
  • To run this file we need to run command docker-compose up
  • If you want to run it in detached mode (in the background), you can use the -d flag: docker-compose up -d
  • When all dependancy installed in conatiner then project going to run and check on localhost:8000
  • If done any local change then we need to run cmd docker-compose up --build Like any migration or any settings changes.
